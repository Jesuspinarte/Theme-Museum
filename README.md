# Theme Museum

**Final RGP Project - Capstone Prototype**

### 🔗 [Play Demo on Itch.io](https://jesuspinarte.itch.io/theme-museum)

## Overview

A complex management simulation inspired by the **Manchester Museum**. Players build galleries, manage finances, and curate world-class exhibits. This is the most technically complex project of the RGP module, integrating building systems with dynamic AI.

## Technical Implementation
- **Grid Building System:** A custom-built system for placing walls and objects with real-time collision and "blueprint" validation.
- **Contextual Visitor AI:** NPCs that decide which objects to admire based on the museum's "Attractiveness Score" and room types.
- **Advanced Tooltip System:** A Raycast-based system that fetches data from ScriptableObjects to display historical context (lore).
- **Multi-Channel Audio:** Interruption-based SFX management for a cleaner user experience during rapid building.

## Controls
- **Left Click:** Place objects/walls and select menu items.
- **Right Click:** Cancel placement/Blueprint mode.
- **Hover:** View object information (Tooltips).
